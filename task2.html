All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value"

The alt Attribute The required alt attribute for the img tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection,or an error in the src attribute, or if the user uses a screen reader.

The title Attribute The title attribute defines some extra information about an element.
